# 00. 레포 부트스트랩

## 목표

미국 주식과 한국 주식을 함께 다루는 리서치 워크스페이스의 기본 구조를 점검한다.

## 참고 문서

- `AGENTS.md`
- `README.md`
- `docs/architecture/page-manifest.yaml`
- `docs/architecture/component-manifest.yaml`
- `docs/design/design-memory.md`

## 제약

- 무한 루프형 자동화와 외부 채팅 브리지 자동화는 사용하지 않는다.
- 실제 데이터 연동을 우선하고, 대체 데이터는 사용자 화면에서 `(목데이터)`로 표시한다.
- 환경 변수와 비밀 값은 커밋하지 않는다.
- 서버가 관리해야 하는 값은 서버 API와 저장소를 통해 다룬다.

## 완료 조건

- 필수 문서를 읽고 현재 구조를 설명할 수 있다.
- 로컬 의존성 설치와 기본 검증 명령을 확인했다.
- 이후 작업의 기준 브랜치와 PR 대상이 명확하다.
