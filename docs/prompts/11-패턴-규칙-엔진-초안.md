# 목표
현재 종목 차트가 어떤 패턴과 유사한지 설명할 수 있는 규칙 기반 초안을 만든다.

# 반드시 읽을 문서
- docs/design/chart-pattern-atlas.md
- docs/design/stock-reference-blueprint.md
- docs/prompts/09-완성형-주식-분석-사이트.md

# 구현 목표
- 이동평균, 최근 고점/저점, 거래량, 박스권, 돌파 여부를 기반으로 유사 패턴 점수를 계산한다.
- LLM 추측이 아니라 먼저 규칙 기반 점수로 후보를 좁힌다.
- 결과는 "패턴명 + 유사 이유 + 무효화 조건" 구조로 반환한다.

# 최소 지원 패턴
- Cup with Handle
- Double Bottom
- Flat Base
- Ascending Base
- Square Box

# 출력 형식 예시
- pattern: "Double Bottom"
- confidence: 0.72
- stage: "넥라인 재도전 구간"
- reasons:
  - "두 번째 저점이 첫 번째 저점보다 높음"
  - "20일선 회복 시도"
  - "거래량 회복은 아직 약함"
- invalidation:
  - "직전 저점 하향 이탈"
- summary:
  - "현재 흐름은 double bottom 유사 구조지만 넥라인 돌파 확인 전이라 관찰 구간으로 보는 편이 안전합니다."

# 주의사항
- 확정 판정처럼 말하지 말 것
- 패턴만으로 매수/매도 판단을 내리지 말 것
- 거래량과 이벤트 타임라인을 함께 고려할 것

# 완료 조건
- 프론트 또는 서버에서 재사용 가능한 규칙 함수가 생긴다
- 최소 3개 이상의 패턴에 대해 테스트 가능한 입력/출력이 있다
- 한국어 설명 요약까지 연결 가능한 형태다
