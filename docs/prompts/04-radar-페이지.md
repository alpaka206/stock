# 04. Radar 페이지

## 목표

관심종목을 빠르게 훑고, 조건 감지와 섹터 흐름을 한 화면에서 판단할 수 있는 레이더 화면을 만든다.

## 기본 구성

- 상단: 검색, 폴더 필터, 정렬, 선택 종목 요약
- 중앙: 경량 테이블 기반 관심종목 목록
- 우측 또는 하단: 조건 감지 알림, 섹터 요약, 다음 액션
- 종목 선택 시 `/stocks/[symbol]`로 자연스럽게 이동

## 제약

- 행과 셀 크기가 hover나 선택 상태에서 흔들리지 않아야 한다.
- 숫자는 등폭 스타일을 사용해 스캔이 쉬워야 한다.
- 알림은 과장된 표현보다 사용자가 확인해야 할 사실을 우선한다.
- 실제 조건 감지 데이터가 없으면 `(목데이터)`로 표시한다.

## 완료 조건

- `/radar`에서 검색, 폴더, 정렬, 종목 이동이 동작한다.
- 조건 감지 카드가 표시되고 출처 상태가 명확하다.
- Playwright 또는 브라우저 확인으로 레이아웃 깨짐이 없다.
