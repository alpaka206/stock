# Chart Pattern Atlas

## 목표
사용자가 차트를 보며 "지금 이 종목이 어떤 모양으로 흘러가고 있는지"를 직관적으로 이해하게 만든다.

중요한 것은 패턴 이미지를 예쁘게 나열하는 것이 아니라, 현재 차트와 패턴 도감을 연결해 설명하는 것이다.

## 제품에서 보여줘야 하는 방식

### 1. 패턴 도감 카드
종목 상세 페이지 또는 차트 우측 패널에서 대표 패턴을 카드형으로 보여준다.

권장 패턴:
- Cup with Handle
- Double Bottom
- Saucer Base
- Flat Base
- Square Box
- Ascending Base
- High Tight Flag
- Three Tight Closes
- Short Stroke
- Continuation / Neutral / Reversal / Special 패턴 요약

카드 구성:
- 패턴 이름
- 단순화한 라인 이미지 또는 도식
- 이 패턴이 보통 어떤 맥락에서 나오는지
- 초보자용 한 줄 설명
- 주의할 무효화 조건

### 2. 현재 차트 해석 카드
패턴 도감과 별개로, 실제 종목 차트 옆에는 아래 문구가 보여야 한다.

예시:
- "현재 차트는 cup with handle과 유사한 구조입니다."
- "아직 손잡이 구간이 짧아 완성형으로 보긴 이릅니다."
- "20일선 위에서 거래량이 줄며 눌리는 흐름이라 돌파 전 관찰 구간입니다."
- "직전 고점을 회복하지 못하면 패턴 완성으로 보기 어렵습니다."

핵심은 패턴명만 던지는 것이 아니라 지금 어느 단계인지 말해주는 것이다.

## UX 원칙

### 초보자용
- 현재 유사 패턴 1개를 크게 보여준다
- "지금 이런 그림에 가까워요" 수준의 쉬운 설명을 먼저 준다
- 어려운 용어는 바로 옆에서 풀어 쓴다

### 숙련자용
- 상위 3개 유사 패턴과 신뢰도
- 거래량 확인 여부
- 이동평균 정렬
- 돌파 후보 가격대
- 무효화 가격대 또는 추세 약화 조건

## 출력 방식

### 좋은 출력 예시
- "현재 구조는 double bottom 유사 패턴입니다. 두 번째 저점이 첫 번째 저점보다 높고, 20일선 회복이 진행 중이라 바닥 다지기 관점에서 볼 수 있습니다."
- "flat base와 square box 사이의 횡보 구조에 가깝습니다. 거래량이 줄며 박스 상단을 테스트하는 구간이라 돌파 확인 전까지는 관찰이 우선입니다."
- "고점 돌파 시도는 있으나 거래량이 충분하지 않아 high tight flag로 보기에는 근거가 약합니다."

### 피해야 할 출력 예시
- "무조건 상승 패턴입니다"
- "곧 급등합니다"
- "지금 반드시 사야 합니다"

## 시각화 방향

### 패턴 도감 패널
- 이미지 카드 6개에서 12개를 한 번에 스캔 가능하게 배치
- 너무 교과서 페이지처럼 보이지 않게 서비스 톤에 맞춘 카드 디자인 사용
- 현재 종목과 가장 비슷한 패턴은 강조 표시

### 현재 차트 비교 뷰
- 왼쪽: 실제 종목 차트
- 오른쪽: 단순화한 패턴 도식
- 아래: 왜 이 패턴과 비슷한지 3줄 설명

### 상태 라벨
권장 상태 라벨:
- "유사 패턴 진행 중"
- "패턴 형성 초기"
- "돌파 관찰 구간"
- "무효화 주의"
- "과열 주의"
- "추세 전환 확인 필요"

## 데이터/해석 원칙
- 패턴 인식은 확정 판정이 아니라 유사도 기반 설명으로 제공
- 이동평균선, 거래량, 최근 고점/저점, 이벤트 타임라인을 함께 봐야 한다
- 패턴만으로 매수/매도 결론을 내리지 않는다
- 뉴스, 공시, 실적 발표 같은 이벤트가 패턴 해석을 바꿀 수 있다는 점을 같이 보여준다

## 구현 체크리스트
- 패턴 도감 카드 UI
- 현재 차트 유사 패턴 카드
- 패턴 유사 이유 설명
- 무효화 조건 설명
- 초보자용 쉬운 해설
- 숙련자용 세부 근거
