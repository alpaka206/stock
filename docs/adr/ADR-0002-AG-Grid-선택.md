# ADR-0002: Radar 핵심 그리드로 AG Grid 선택

- 상태: 승인
- 날짜: 2026-03-15

## 배경
Radar 화면은 단순 리스트가 아니라 다음 기능을 중심으로 동작한다.
- 엑셀처럼 밀도 높은 셀 UX
- 컬럼 토글
- 그룹 뷰
- 커스텀 셀
- 향후 대량 데이터/서버사이드 확장

## 결정
기본 그리드 라이브러리로 AG Grid 를 선택한다.

## 이유
- headless table 보다 즉시 사용 가능한 고급 grid 기능이 많다.
- 사용자 요구사항이 "표"보다 "작업용 grid"에 가깝다.
- 향후 고급 기능 확장을 위한 고점이 높다.

## 주의
- row grouping, cell selection 같은 일부 핵심 UX는 Enterprise 기능 여부를 먼저 확인하고 라이선스 비용을 고려한다.
