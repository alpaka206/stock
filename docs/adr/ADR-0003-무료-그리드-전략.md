# ADR-0003: 무료 그리드 전략으로 AG Grid Community 고정

- 상태: 승인
- 날짜: 2026-03-15

## 배경
Radar 화면은 grid 중심 인터랙션이 필요하지만 현재 단계에서는 유료 라이선스 비용을 전제하지 않는다.

## 결정
그리드 구현은 AG Grid Enterprise 가 아니라 AG Grid Community 범위 안에서 설계한다.

## 이유
- 초기 단계에서 비용과 라이선스 부담 없이 dense grid UX 를 검증할 수 있다.
- 정렬, 필터, 커스텀 셀, 상태 저장 같은 핵심 기능은 Community 만으로도 충분히 시작 가능하다.
- Enterprise 의존을 피하면 문서와 구현이 더 명확해진다.

## 영향
- row grouping, pivot, columns tool panel 같은 Enterprise 기능에 직접 의존하지 않는다.
- 폴더 트리, view mode, 커스텀 컬럼 토글로 대체 UX 를 설계한다.
- Radar 구현과 문서 전반에서 무료 범위 기준을 유지해야 한다.

## 단점 / 트레이드오프
- Enterprise 수준의 고급 그룹 UX 는 직접 조합해야 한다.
- 컬럼 관리와 그룹 뷰는 커스텀 컴포넌트 유지보수 비용이 생긴다.

## 후속 작업
- Radar 의 컬럼 토글과 그룹 뷰를 공통 컴포넌트로 정리한다.
- 라이선스 정책이 바뀌면 별도 ADR 로 재평가한다.
