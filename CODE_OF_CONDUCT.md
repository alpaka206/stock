# 커뮤니티 운영 원칙

이 저장소는 협업 가능한 공개 프로젝트를 지향합니다. 참여자는 아래 기준을 따라야 합니다.

## 기대하는 태도

- 기술적 반박은 가능하지만 사람을 공격하지 않습니다.
- 재현 정보와 근거를 중심으로 대화합니다.
- 초보자 질문도 무시하지 않고 필요한 전제를 설명합니다.
- 시장/투자 관련 내용은 정보 공유로만 다루고, 확정 수익을 보장하는 표현은 지양합니다.

## 허용하지 않는 행동

- 모욕, 비하, 괴롭힘, 혐오 표현
- 스팸, 광고성 홍보, 도배
- 개인정보, 비밀키, 토큰, 계정 정보 노출
- 취약점을 공개 이슈나 공개 Discussion에 게시하는 행위

## 운영 기준

- 저장소 관리자는 커뮤니티 안전과 코드 품질을 위해 댓글, 이슈, PR을 편집·잠금·삭제할 수 있습니다.
- 반복 위반은 차단 사유가 됩니다.

## 신고

- 일반 운영 문제: GitHub Discussions 또는 이슈
- 보안 문제: [`.github/SECURITY.md`](.github/SECURITY.md) 절차 사용
